# LLZX_MakingWheels

这是一个用于学习和实践各种基础组件的代码仓库，包含缓存系统、内存池等常见的底层组件实现。本仓库的主要目的是通过自行实现这些组件来加深对计算机系统底层原理的理解。

## 目录结构

```
LLZX_MakingWheels/
├── CMakeLists.txt       # 顶层CMake配置文件
├── build/               # 构建输出目录
├── cache_system/        # 缓存系统项目
│   ├── CMakeLists.txt   # 缓存系统CMake配置
│   ├── include/         # 头文件目录
│   └── src/             # 源代码目录
├── memory_pool/         # 内存池项目
│   ├── include/         # 头文件目录
│   └── src/             # 源代码目录
├── common/              # 公共工具库
│   ├── include/         # 头文件目录
│   └── src/             # 源代码目录
└── README.md            # 项目说明文档
```

## 项目说明

### 学习目标

本仓库旨在帮助你：
- 深入理解常见底层组件的实现原理
- 掌握C++语言的高级特性和编程技巧
- 学习如何设计和实现高性能的系统组件
- 练习使用CMake进行项目管理

### 项目组件

1. **缓存系统 (cache_system)**
   - 实现各种缓存淘汰策略（如LRU、LFU等）
   - 学习缓存设计原理和性能优化

2. **内存池 (memory_pool)**
   - 实现高效的内存分配器
   - 学习内存管理原理和减少内存碎片

3. **公共工具库 (common)**
   - 提供项目中使用的通用工具函数和类

## 使用方法

### 构建项目

1. 进入build目录：
   ```bash
   cd build
   ```

2. 配置CMake：
   ```bash
   cmake ..
   ```

3. 编译项目：
   ```bash
   make
   ```

### 扩展项目

如果你想添加新的组件项目，请按照以下步骤：

1. 创建新的项目目录：
   ```bash
   mkdir -p new_component/include new_component/src
   ```

2. 在新目录中创建CMakeLists.txt文件

3. 在顶层CMakeLists.txt中添加新组件的构建选项
